UpdateTempCatalogAsync()

Downloads the default catalog into memory without replacing the active catalog. Use this to check download sizes (via GetDownloadSize* methods) without affecting currently loaded assets. Returns true if a newer catalog exists on the remote, false if the catalog is already up to date. The temporary catalog is automatically cleared when UpdateCatalogAsync is called.

public static TAsyncOperation<bool> UpdateTempCatalogAsync()

Events

OnCatalogLoaded

Raised after the active catalog is (re)loaded into memory — both on successful InitializeAsync() and on successful UpdateCatalogAsync(). Subscribers should treat any previously cached asset metadata (including path resolution caches) as stale and refresh as needed.
